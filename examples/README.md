# portfolio-media Examples

This directory contains usage examples for the `portfolio-media` media generation toolkit.

## Examples

### `basic_usage.py`
Basic logo generation for a Python package.

Run:
```bash
cd packages/portfolio-media
python examples/basic_usage.py
```

### `video_generation.py` (TODO)
Generating explainer videos with PBS NOVA style.

### `batch_generation.py` (TODO)
Batch media generation for multiple packages.

## Setup

Before running examples, ensure you have:
1. Python 3.12+
2. Required dependencies installed (`pip install -e .`)
3. API keys configured (if using AI-powered generation)

## Expected Output

For `basic_usage.py`:
```
Logo generated: assets/logos/my-awesome-package-logo.png
Dimensions: 512x512
Format: PNG
Size: 45678 bytes
```
